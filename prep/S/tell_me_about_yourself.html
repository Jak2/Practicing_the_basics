Project story 1 — AI & semantic models (40 sec)

"One of the things I'm most proud of is that I was doing AI-augmented development before most organisations even had a policy for it. I built a workflow using GitHub Copilot connected to an MCP Server — that's a Model Context Protocol server — integrated directly into VS Code, which allowed me to use natural language to create and modify Power BI semantic models programmatically. Tables, DAX measures, relationships, security roles — all built through AI-assisted natural language commands on a PBIP text-based architecture with full Git versioning. Microsoft themselves published this as their recommended enterprise workflow, and I had it running in production."

Project story 2 — navigation framework (35 sec)

"I also engineered a Storyboard URL framework in Power BI that connected ten to twelve dashboards through Investment-ID-based drill-through and dynamic DAX-driven URL routing. This reduced navigation time by 25 to 40 percent, eliminated manual lookups entirely, and saved the team three to five hours every single week. The key was metadata-aligned modelling — ensuring 100 percent data consistency across every dashboard, so when a finance leader and an engineering leader look at the same investment number in different reports, they see exactly the same figure."

Project story 3 — strategic leadership dashboards (40 sec)

"I designed two dashboards that directly touched leadership strategy. First, the ZBB-PP2026 dashboard — a 360-degree view of Eaton's enterprise roadmap covering finance, resource allocation, investment details, and baseline comparison. This improved strategic planning accuracy by 15 percent and leadership decision-making efficiency by 20 percent. Second, the Honeycomb dashboard, which mapped engineering deliverables against six of Eaton's strategic priorities — including NPI velocity and sustainability KPIs — and became the single source of truth for cross-functional performance. Business leaders specifically recognised it for its UX design and real-time reliability."

Project story 4 — Python ETL pipeline (35 sec)

"Beyond dashboards, I built a centralised data integration layer — the CSPP Software Excellence Matrix — that automated extraction from GitHub, Jira, and Clarity PPM using Python API pipelines, and unified them through Power BI Dataflows with a robust ETL architecture. Before this, these three platforms had completely fragmented, siloed data. After, any cross-functional team could query engineering performance, project status, and resource allocation from a single certified model — with zero manual effort and zero reporting errors."
